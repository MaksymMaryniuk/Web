HTML (HyperText Markup Language) – це мова розмітки, що використовується для створення веб-сторінок і визначення їх структури та зовнішнього вигляду. Вона складається з набору тегів, які використовуються для позначення різних елементів веб-сторінки, таких як заголовки, параграфи, таблиці, зображення і посилання. HTML є основною мовою для створення веб-сторінок, і вона використовується разом з CSS (Cascading Style Sheets) і JavaScript для створення багатофункціональних інтерактивних веб-сайтів. CSS використовується для задання стилів і зовнішнього вигляду елементів HTML, а JavaScript додає можливості динамічного змінення контенту сторінки та взаємодії з користувачем. HTML використовує просту синтаксичну структуру, яка складається з тегів, атрибутів і значень. Теги відкриваються за допомогою “<” і закриваються за допомогою “>”, при цьому вміщений між ними контент відображається на веб-сторінці. Наприклад, тег “p” використовується для позначення параграфу, тег “img” – для вставки зображень, а тег “a” – для створення посилань. Основна ідея HTML полягає в тому, що вона створює структуру документа, яку веб-браузери можуть інтерпретувати та відображати користувачу. Вона дозволяє розміщувати текст, зображення, відео, звук і інші мультимедійні елементи на веб-сторінці і забезпечує їх відображення відповідним чином.
Що таке CSS? Кожен HTML-документ має базові параметри для тегів. Наприклад, розташування параграфів для тега [p], точки для ненумерованих списків і лінії розмітки для табличок. Але сьогодні цього недостатньо. Сайт має бути привабливим, красиво оформленим і наповнений графічними елементами.
Frontend React для чого потрібен? React використовується для розробки користувацького інтерфейсу веб-додатків. Основною метою React Frontend є створення інтерактивних, динамічних та відзивчивих інтерфейсів для користувачів. React Front end дозволяє створювати багатофункціональні та інтерактивні застосунки зі швидким рендерингом і перехідом між сторінками.
